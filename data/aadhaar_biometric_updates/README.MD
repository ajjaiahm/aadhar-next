# Aadhaar Biometric Update Dataset

This dataset contains aggregated statistics related to biometric updates
in Aadhaar, including Fingerprint, Iris, and Face modalities.

Purpose:
- Track periodic biometric revalidation
- Analyze child-to-adult biometric transition
- Study biometric failure and refresh patterns

Privacy:
- Aggregated, anonymised data
- No Aadhaar numbers or personal identifiers

Modalities:
- Fingerprint
- Iris
- Face

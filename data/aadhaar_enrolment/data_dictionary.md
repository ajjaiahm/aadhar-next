# Aadhaar Enrolment Dataset – Data Dictionary

| Column Name     | Description |
|-----------------|-------------|
| date            | Enrolment date (YYYY-MM or YYYY-MM-DD) |
| state           | State name |
| district        | District name |
| pincode         | PIN code |
| age_0_5         | Number of enrolments (0–5 years) |
| age_5_17        | Number of enrolments (5–17 years) |
| age_18_plus     | Number of enrolments (18+ years) |

Notes:
- Dataset is aggregated and anonymised
- No personal or biometric data is present
- Data provided by UIDAI for analytics purposes

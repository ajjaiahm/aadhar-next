# Setup Guide

This repo contains 3 components:
1) Go Auth Gateway (port 8080)
2) Python ML Engine (FastAPI, port 8000)
3) Next.js Dashboard (port 3000/3001)

## Prerequisites
- Git
- Go (>= 1.20 recommended)
- Python (>= 3.10)
- Node.js + npm (LTS recommended)

## Clone & Branch
Do NOT work on main. Work on a feature branch.

Example:
```bash
git clone -b aj_works https://github.com/ajjaiahm/aadhar-next.git
cd aadhar-next
git checkout -b <your-branch>
